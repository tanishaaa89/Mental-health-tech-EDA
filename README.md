# Mental Health in Tech — EDA & HR Analytics Dashboard

## Overview
End-to-end data analysis of the OSMI Mental Health in Tech Survey 
(1,251 respondents, 26 variables) using Python and Power BI.

## Key Findings
- 50.5% of tech employees have sought mental health treatment
- Female employees seek treatment at 68.8% vs 45.4% for males — 
  a 23-point gender gap suggesting stigma among men in tech
- "Sometimes" is the most common work interference level (37%) — 
  widespread but underdiscussed mental health impact in the industry

## Tools & Technologies
Python · Pandas · NumPy · Matplotlib · Seaborn · Plotly · Power BI · DAX
